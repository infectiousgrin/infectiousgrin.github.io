Here is a Filter function for a slice of integer numbers. It goes over all the items in the slice and returns a new array containing items that pass a test you specify.

// Filter returns a new slice containing all strings in the slice that satisfy
// the predicate f.
func Filter(vs []int, f func(int) bool) []int {
    vsf := make([]string, 0)
    for _, v := range vs {
        if f(v) {
            vsf = append(vsf, v)
        }
    }
    return vsf
}

For example, given the following slice:

numbers := []int{1, 2, 4, 7, 9, 12, 13}

We could use Filter() to create an array of all the numbers that are greater than or equal to 5:

over5 := Filter(numbers, func(v int) bool {
  return v >= 5
})

Alternatively, we could use Filter() to find odd numbers using modulus:

odd := Filter(numbers, func(v int) bool {
  return v % 2 == 1
})
